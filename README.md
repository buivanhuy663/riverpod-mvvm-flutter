# Riverpod MVVM Flutter

Boilerplate for flutter app

- State management : rivepod
- Dependence : riverpod
- Architecture : MVVM

## Getting Started

1. Install FVM if you want (https://fvm.app/documentation/getting-started/installation)
2. Run `flutter pub get`
3. Launch
